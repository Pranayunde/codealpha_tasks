* Product Listings

* Shopping Cart

* Authentication & Authorization
* Responsive Design
* Database Integration

* HTML5
* CSS3
* JavaScript
* Node.js
* Express.js
* SQLite
* JWT Authentication

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/Pranayunde/codealpha_tasks.git
```

### Navigate to the Repository

```bash
cd codealpha_tasks
```

### Run Any Project

Navigate to the desired project folder:

```bash
cd Ecommerce-Project
```

or

```bash
cd SocialMedia-Project
```

Install dependencies:

```bash
npm install
```

Start the application:

```bash
npm start
```
